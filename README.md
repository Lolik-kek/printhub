# PrintHub

Онлайн-сервис заказа услуг 3D-печати: каталог материалов и принтеров, калькулятор стоимости печати, корзина и оформление заказов, личный кабинет, новости отрасли и панель администрирования.

## Технологии

- HTML, CSS, JavaScript (статический многостраничный сайт)
- Supabase (PostgreSQL, Auth, RLS) — каталог, заказы и авторизация

## Структура

- `pages/` — страницы сайта (главная, материалы, принтеры, новости, калькулятор, корзина, профиль, вход, регистрация, контакты, администрирование)
- `css/` — стили
- `js/` — клиентская логика (каталог, корзина, калькулятор, авторизация, администрирование)
- `images/` — изображения каталога

## Возможности

- Каталог материалов и принтеров с фильтрацией и сортировкой
- Онлайн-калькулятор стоимости 3D-печати
- Корзина и оформление заказа
- Регистрация и авторизация пользователей
- Личный кабинет с историей заказов
- Лента новостей о 3D-печати
- Панель администрирования каталогом и заказами
